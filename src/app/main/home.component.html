Bienvenido a nuestra aplicación

Esta es una aplicación pensada para ir anotando aquellos pequeños deseos o caprichos que queremos adquirir pero qué no sabemos exactemente qué modelo, marca o dónde adquirirlo.

Registrándote, con esta aplicación podrás ir registrándolos en una lista de deseos y, para cada uno de ellos, ir recopilando las diferentes opciones, para posteriormente decidirnos por la más convenientes.

Si ya tienes cuenta, loguéate. Si no, crea tu cuenta ya.
